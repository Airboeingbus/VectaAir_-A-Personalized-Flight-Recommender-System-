About This System

🎯 How It Works

This hybrid recommender system combines:

  • User Similarity Graph: Finds users with similar preferences
  • Weighted Scoring: Uses interaction patterns (bookings, views)
  • Flight Reliability: Considers on-time performance data

📈 Scoring Formula

Score = Σ(similarity × weight) × reliability

⭐ Score Interpretation

0.90–1.0 Excellent
0.75–0.89 Good
0.50–0.74 Fair
Below 0.50 Consider Other Options

💡 Tips

  • Try different User IDs (U00001, U00002, etc.)
  • Check the score to assess recommendation quality
  • Higher scores = better match + more reliable flight
